how to add CSS to HTML?
there are 3 ways to add CSS to HTML
1. Inline CSS
- adding styles inside HTML tags with style attribute
    <h1 style="text-align: center;"></h1>

2. Internal CSS
- adding styles inside the head tag 
<style>
    h1 {
        text-align: center;
    }
</style>
